A RESTful API for tracking job applications and interviews built with Java and Spring Boot.

Tech Stack
- Java 21 | Spring Boot 3.4.3 | MySQL | Spring Security | JWT | Maven

## Features
- Register and login with JWT authentication
- Track job applications with status (APPLIED, INTERVIEW, OFFER, REJECTED)
- Log interviews with notes and outcomes
- Save recruiter contact details
- Role based access control (ADMIN, USER)

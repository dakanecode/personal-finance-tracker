 Personal Finance Tracker API

A RESTful API for managing personal finances built with Java and Spring Boot.
Track your income, expenses and budgets — all secured with JWT authentication.

 Tech Stack
- Java 21 | Spring Boot 3.4.3 | MySQL | Spring Security | JWT | Maven

## Features
- Register and login with JWT authentication
- Track income and expense transactions
- Categorize transactions (Food, Transport, Salary etc)
- Set monthly budget limits per category
